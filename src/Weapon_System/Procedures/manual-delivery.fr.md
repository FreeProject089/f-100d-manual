# Livraison manuelle

Pour des profils de livraison spécifiques, consultez les [tableaux de lancement d'armes](../bomb-rockets.md)

## Fusées

La procédure ci-dessous configurera les armes de manière à ce que le bouton de la bombe tire une salve de roquettes.

!!! note
    Les roquettes sont plus efficaces dans les modes de tir ondulé.

### 1. Armement des fusées

1. Panneau de commande d'armement : [Sélecteur de magasins](../armament-control-panels.md#stores-selector)—**RKTS**
2. Panneau de commande d'armement : [Sélecteur de mode](../armament-control-panels.md#mode-selector)—**MANUEL**
3. Régler tous les panneaux de commande d'armement : [Commutateurs de sélection de libération de station](../armament-control-panels.md#station-release-selection-switches)—**NORM**

### 2. Configuration du système de largage d'armes d'avion (AWRS)

1. Définir AWRS [Sélecteur de quantité](../aircraft-weapons-release-system.md#controls)—**COMME DÉSIRÉ**
2. Définir AWRS [Intervalle de libération](../aircraft-weapons-release-system.md#controls)—**COMME DÉSIRÉ**—(50 ms recommandés)
3. Définir AWRS [Sélecteur de mode](../aircraft-weapons-release-system.md#controls)—**RIPPLE SALVO**


### 3. Configuration de la vue

1. Réglez le [Sélecteur de visée](../a-4-gunsight.md#1-sight-selector-switch) —**ROCKET**
2. Réglez [Dépression de la vue](../a-4-gunsight.md#2-sight-depression)—**COMME DÉSIRÉ**—(10-20 mils recommandés)
3. Définir [Levier de cage mécanique](../a-4-gunsight.md#mechanical-cage-lever)—**UNCAGE**

## Bombes

La procédure ci-dessous configurera les armes de manière à ce que le bouton de bombe largue une série de bombes.

!!! note
    Les bombes sont plus efficaces dans les modes de tir ondulé.

### 1. Armement des bombes

1. Panneau de contrôle de l'armement : [Sélecteur de magasins](../armament-control-panels.md#stores-selector)—**BOMBS**
2. Panneau de commande d'armement : [Sélecteur de mode](../armament-control-panels.md#mode-selector)—**MANUEL**
3. Régler tous les panneaux de commande d'armement : [Commutateurs de sélection de libération de station](../armament-control-panels.md#station-release-selection-switches)—**NORM**

4. Définir tout le panneau de commande d'armement : [Commutateur de bras de bombe](../armament-control-panels.md#bomb-arm-switch)—**NOSE & TAIL**

!!! note
    Si vous déposez une traînée élevée en mode faible traînée, réglez plutôt le [Commutateur de bras de bombe](../armament-control-panels.md#bomb-arm-switch) sur **NOSE**.

### 2. Configuration du système de largage d'armes d'avion (AWRS)

1. Définir AWRS [Sélecteur de quantité](../aircraft-weapons-release-system.md#controls)—**COMME DÉSIRÉ**
2. Définir AWRS [Intervalle de libération](../aircraft-weapons-release-system.md#controls)—**COMME DÉSIRÉ**—(150 ms recommandés)
3. Définir AWRS [Sélecteur de mode](../aircraft-weapons-release-system.md#controls)—**RIPPLE SINGLE**

### 3. Configuration de la vue

1. Réglez le [Sélecteur de visée](../a-4-gunsight.md#1-sight-selector-switch)—**BOMBE**
2. Définir la [Dépression de la vue](../a-4-gunsight.md#2-sight-depression)—**TEL QUE REQUIS**—(Voir les [tableaux de lancement d'armes](../bomb-rockets.md))
3. Définir [Levier de cage mécanique](../a-4-gunsight.md#mechanical-cage-lever)—**UNCAGE**
