# Manual Delivery

## Rockets

The procedure below will setup the weapons so that the bomb-button will fire a salvo of rockets.

### 1. Rocket Arming

1. Armament Control Panel: [Stores Selector](../armament-control-panels.md#controls) - RKTS
2. Armament Control Panel: [Mode Selector](../armament-control-panels.md#controls) - MANUAL
3. Set All Armament Control Panel: [Station Release Selection Switches](../armament-control-panels.md#controls) - NORM

### 2. Aircraft Weapons Release System (AWRS) Setup

It is suggested to ripple fire rockets - this is the what this procedure will describe.

1. Set AWRS [Quantity Selector](../aircraft-weapons-release-system.md#controls) - desired number of salvos
2. Set AWRS [Release Interval](../aircraft-weapons-release-system.md#controls) - desired interval (50 ms if no preference)
3. Set AWRS [Mode Selector](../aircraft-weapons-release-system.md#controls) - RIPPLE SALVO.

### 3. Sight Setup

1. Set [Sight Selector Switch](../a-4-gunsight.md#1-sight-selector-switch) - ROCKET.
2. Set [Sight Depression](../a-4-gunsight.md#2-sight-depression) - desired depression (otherwise about 10-20 mils)
3. Set [Mechanical Cage Lever](../a-4-gunsight.md#mechanical-cage-lever) - UNCAGE

## Bombs

The procedure below will setup the weapons so that the bomb-button will drop a series of bombs.

### 1. Bomb Arming

1. Armament Control Panel: [Stores Selector](../armament-control-panels.md#controls) - BOMBS
2. Armament Control Panel: [Mode Selector](../armament-control-panels.md#controls) - MANUAL
3. Set All Armament Control Panel: [Station Release Selection Switches](../armament-control-panels.md#controls) - NORM

4. Set All Armament Control Panel: [Bomb Arm Switch](../armament-control-panels.md#controls) - NOSE & TAIL (NOSE only if using high drag bombs to be dropped in low drag mode)

### 2. Aircraft Weapons Release System (AWRS) Setup

It is suggested to ripple fire rockets - this is the what this procedure will describe.

1. Set AWRS [Quantity Selector](../aircraft-weapons-release-system.md#controls) - desired number of bombs
2. Set AWRS [Release Interval](../aircraft-weapons-release-system.md#controls) - desired interval (otherwise about 150 ms)
3. Set AWRS [Mode Selector](../aircraft-weapons-release-system.md#controls) - RIPPLE SINGLE.

### 3. Sight Setup

1. Set [Sight Selector Switch](../a-4-gunsight.md#1-sight-selector-switch) - BOMB.
2. Set [Sight Depression](../a-4-gunsight.md#2-sight-depression) - desired depression (see depression chart)
3. Set [Mechanical Cage Lever](../a-4-gunsight.md#mechanical-cage-lever) - UNCAGE
