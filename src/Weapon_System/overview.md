# Weapons System Overview

## Introduction

The F-100D weapon system is split into two major sections with some minor overlap: air-to-air and air-to-ground.

## Air-to-Air

Air-to-air systems include.

- [missile control panel](./aim-9-sidewinder.md#controls) used for air-to-air sidewinder missiles.
- [A4 Gunsight](./a-4-gunsight.md) used for air-to-air gunnery and missile operation.
- [Gun/Missile Selector Switch]
- [Gun Camera](../Gun%20Camera/gun_camera.md) 


## Air-to-Ground

The aircraft weapon release circuitry is split into two major sections. The first being the weapon selection circuitry this determines which pylons are selected for release based on weapon type and weapon quantity. The second section are the release systems these generate release pulses for the selected pylons.
