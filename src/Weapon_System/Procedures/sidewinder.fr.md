# Sidewinder

1. [Maître de missile](../aim-9-sidewinder.md#missile-master)—PRÊT
2. [Commutateur de missiles à canon](../guns.md#gun-missile-switch)—MISSILES
3. Vérifiez que le voyant d'état du missile est vert (sinon, utilisez [Station Bypass](../aim-9-sidewinder.md#station-bypass) pour avancer jusqu'à ce qu'un missile fonctionnel puisse être trouvé, la gâchette peut également être utilisée pour avancer mais risque de tirer par inadvertance un sidewinder). Le ton doit être entendu.
4. Écoutez une tonalité plus aiguë indiquant une source de chaleur valide et appuyez sur la gâchette du deuxième étage pour tirer le missile.
