Manual Delivery

For specific delivery profiles see the weapon release tables

Rockets

The procedure below will setup the weapons so that the bomb-button will fire a salvo of rockets.

1. Rocket Arming

  1. Armament Control Panel: Stores Selector - RKTS
  2. Armament Control Panel: Mode Selector - MANUAL
  3. Set All Armament Control Panel: Station Release Selection Switches - NORM

2. Aircraft Weapons Release System (AWRS) Setup

It is suggested to ripple fire rockets - this is the what this procedure will describe.

  1. Set AWRS Quantity Selector - desired number of salvos
  2. Set AWRS Release Interval - desired interval (50 ms if no preference)
  3. Set AWRS Mode Selector - RIPPLE SALVO.

3. Sight Setup

  1. Set Sight Selector Switch - ROCKET.
  2. Set Sight Depression - desired depression (otherwise about 10-20 mils)
  3. Set Mechanical Cage Lever - UNCAGE

Bombs

The procedure below will setup the weapons so that the bomb-button will drop a series of bombs.

1. Bomb Arming

  1. Armament Control Panel: Stores Selector - BOMBS
  2. Armament Control Panel: Mode Selector - MANUAL

  3. Set All Armament Control Panel: Station Release Selection Switches - NORM

  4. Set All Armament Control Panel: Bomb Arm Switch - NOSE & TAIL (NOSE only if using high drag bombs to be dropped in low drag mode)

2. Aircraft Weapons Release System (AWRS) Setup

It is suggested to ripple fire rockets - this is the what this procedure will describe.

  1. Set AWRS Quantity Selector - desired number of bombs
  2. Set AWRS Release Interval - desired interval (otherwise about 150 ms)
  3. Set AWRS Mode Selector - RIPPLE SINGLE.

3. Sight Setup

  1. Set Sight Selector Switch - BOMB.
  2. Set Sight Depression - desired depression (see depression chart)
  3. Set Mechanical Cage Lever - UNCAGE
